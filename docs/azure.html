Hosting on Azure Cloud

Set Up Azure Cloud Account


Sign Up Here

Login in Command Line Terminal

To use our Azure cloud, we need to login to our Azure account.

Enter this command to login to Azure.

docker login azure

After entering this command, we will be prompted to login with our browser.

Create an ACI

To use our Azure cloud, we need to create an ACI.

Enter this command to create ACI.

docker context create aci aci_name

Push Image to Dockerhub

Before we push image to docker, we must login to our account using command line.

Enter this command to login

docker login

Then, we need to push our image to docker hub.

Enter this command to login

docker push image_name

Pull Image from Docker Hub to Azure Cloud

Before we push image to docker, we must use our ACI.

Enter this command to use ACI

docker context use aci_name

Then, we need to run our image on our Azure Cloud.

Enter this command to host our image on Azure Cloud

docker run --domainname unique_domain_name -dp 80:80 dockerhub_userID/repository_name

See our Website Hosted on IP address

Now, we can view our website on an IP address.

Enter this command to view our IP address

docker ps

We will see that our image is now hosted.

Our IP address will be under the Ports section.
