Command Line

Basic Commands

Navigation

We need to use command line to navigate our files and directories. This way, we can use important Git and Docker functions in the appropriate location.

This command shows us what our current working directory is.

pwd

To see what files and directories are in our current location, enter the following command.

ls

By seeing what files and directories are in our current location, we can then navigate into those locations using the change directory command. To change to a specific directory enter cd followed by the directory name.

cd directory

We can also enter cd followed by hitting the tab button twice. This would show us which directories we can enter.

We can also return to our home directory by just entering cd by itself. We can also move back to our previous directory using this command.

cd ..

Making Changes

To make a directory, use the command below.

mkdir directory_name

To make an empty file, use the command below.

touch file_name

To rename a file, use the command below.

mv file_name new_name

We can also move a file to a new location using this command.

mv file_name directory

Lastly, we can both move and rename a file at the same time.

mv file_name directory/new_name
